A plugin that helps you identify and delete unused images from the WordPress media library.

== Description ==

This plugin identifies unused images in the media library and allows you to delete them manually.

== Installation ==

1. Upload the plugin to your WordPress site's plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Unused Images' in the dashboard to see and manage unused images.
